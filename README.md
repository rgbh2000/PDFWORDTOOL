# PDF Word Tool

一款功能强大的文档转换工具，支持PDF、Word文档和图片之间的相互转换以及图片压缩功能。

## 开始使用

本项目是一个跨平台的文档处理应用程序，支持以下功能：

- PDF转图片
- 图片转PDF
- Word转PDF
- PDF转Word
- 图片压缩

## 功能介绍

### 1. PDF转图片 (PDF to Images)

将PDF文档的每一页转换为图片格式，方便查看和分享。

### 2. 图片转PDF (Images to PDF)

将多张图片合并成一个PDF文档，便于整理和传输。

### 3. Word转PDF (Word to PDF)

将Word文档转换为PDF格式，保持原有格式不变。

### 4. PDF转Word (PDF to Word)

将PDF文档转换为可编辑的Word文档。

### 5. 图片压缩 (Image Compression)

压缩图片大小，在保证质量的前提下减小文件体积。

## 如何使用

1. 启动应用
2. 在左侧导航栏选择需要的功能
3. 选择输入文件
4. 设置输出路径
5. 点击转换按钮
6. 完成后可在指定目录查看结果文件

## Features

### 1. PDF to Images

Convert each page of a PDF document to image format for easy viewing and sharing.

### 2. Images to PDF

Combine multiple images into a single PDF document for easy organization and transfer.

### 3. Word to PDF

Convert Word documents to PDF format while preserving the original layout.

### 4. PDF to Word

Convert PDF documents to editable Word documents.

### 5. Image Compression

Compress image sizes to reduce file volume while maintaining quality.

## How to Use

1. Launch the application
2. Select the desired function from the left navigation panel
3. Choose the input file(s)
4. Set the output path
5. Click the convert button
6. Check the result files in the specified directory after completion

## 打包说明 (Build Instructions)

### Windows 平台打包

要在 Windows 平台上构建此应用程序，请按照以下步骤操作：

1. 确保已安装 Flutter SDK 和相关环境
2. 在项目根目录下运行以下命令：
   ```bash
   flutter build windows
   ```
3. 构建完成后，可执行文件将位于 build\windows\x64\runner\Release 目录中


## Build Instructions

### Building for Windows

To build this application for Windows platform, follow these steps:

1. Ensure Flutter SDK and related environments are installed
2. Run the following command in the project root directory:
   ```bash
   flutter build windows
   ```
3. After building, the executable will be located in build\windows\x64\runner\Release directory
